# 🤖 Desafio: Controle de Estoque do Mercado

**Contexto:** Um mercado deseja identificar rapidamente quais produtos precisam ser repostos.
**O que fazer:** Crie um vetor de objetos contendo pelo menos 8 produtos. Cada produto deve possuir *nome*, *estoque* e *estoqueMinimo*.

Exiba:
* Quantos produtos precisam de reposição.
* Os nomes desses produtos.
* A porcentagem de produtos que estão abaixo do estoque mínimo.
