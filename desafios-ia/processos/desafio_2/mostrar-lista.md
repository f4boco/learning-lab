# Mostrar Lista de Espera

## Objetivo

Exibir a lista de Espera na seção de saída

## Início

O site é carregado, uma pessoa é adicionada ou a próxima pessoa é chamada

## Fluxo

DECISÃO:
1. O sistema verifica se o vetor está vazio
SIM:
Exibe: "Não há fila de espera"
FIM.
   
NÃO:
2. O sistema percorre o vetor da lista de espera concatenando em um string a senha e o 1º nome da pessoa e o CPF com **
FIM.

## Saídas

* String com a lista
