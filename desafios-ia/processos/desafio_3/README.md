# 🤖 Desafio: O Filtro de Produtos Premium

O código começa com um vetor de objetos contendo produtos de uma loja virtual. Cada objeto possui: nome, preco e categoria. O script deve processar esse vetor para criar uma nova lista contendo apenas os produtos que pertencem à categoria, que o usuário selecionar E que custam a partir do preço que ele digitar.

Resultado: Exiba a lista final filtrada na tela formatando a saída com Template Literals (ex: Produto: ${prod.nome} - Preço: R$ ${prod.preco}).
