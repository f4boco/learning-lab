# 🤖 Desafio: O Extrator de Preços em Texto

Um sistema recebe uma mensagem de texto informal descrevendo o orçamento de um serviço (ex: "O motor custa R$ 450 e a mão de obra ficou por R$ 120").
Seu script deve analisar essa string e extrair todos os valores numéricos que vêm logo após o símbolo R$ .
Resultado: Exiba os valores financeiros encontrados na tela.
