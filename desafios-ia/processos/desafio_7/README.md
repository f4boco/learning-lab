# 🤖 Desafio: O Buscador de Hashtags

Receber o texto de uma publicação de rede social enviada pelo usuário. O script deve analisar o texto, localizar todas as palavras que começam com o caractere # (hashtags) e exibi-las em uma lista na tela.

Resultado: Mostre todas as hashtags detectadas no texto.
