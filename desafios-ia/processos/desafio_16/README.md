# 🤖 Desafio: Formatação de Log de Eventos

Contexto: Um sistema de segurança registra tentativas de login. O formato original da data do sistema não é amigável para o usuário final brasileiro.
O que fazer: Dado o objeto de log abaixo, utilize seus conhecimentos de métodos de data e Template Literals para transformar a data no formato brasileiro (DD/MM/AAAA às HH:MM) e exiba a string final no console.
```JavaScript
const logSeguranca = {
  usuario: "admin_master",
  dataHora: new Date("2026-07-02T14:35:00") // 02 de Julho de 2026, às 14:35
};
// Resultado esperado no console: "O usuário admin_master tentou acessar o sistema em 02/07/2026 às 14:35."
```
