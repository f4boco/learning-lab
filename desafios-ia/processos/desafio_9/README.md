# 🤖 Desafio: O Otimizador de Links (Remover Protocolo)

O programa recebe uma URL completa digitada pelo usuário (ex: https://www.google.com ou http://www.exemplo.com.br). Seu script deve identificar se o link começa com http:// ou https:// e remover essa parte do texto, mantendo apenas o restante do endereço.

Resultado: Exiba a URL limpa na tela (ex: www.google.com).
