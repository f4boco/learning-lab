# 🤖 Desafio: O Filtro de Validade de Produtos

Contexto: Você está trabalhando no sistema de um supermercado e precisa listar os produtos que já venceram para que sejam retirados das prateleiras imediatamente.

O que fazer: Escreva o fluxo lógico para filtrar e exibir uma lista apenas com os nomes dos produtos cuja dataValidade seja anterior à data atual.
