# 🤖 Desafio: Contador de Palavras Específicas

O programa recebe um texto longo e uma palavra a ser contada inserida pelo usuário. O script deve contar exatamente quantas vezes a palavra (independentemente de estar escrita em maiúsculas, minúsculas ou misturada) aparece ao longo de todo o texto.

Resultado: Exiba o número total de ocorrências encontradas.
