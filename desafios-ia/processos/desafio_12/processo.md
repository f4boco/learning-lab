# Processo do site

## Objetivo

Remover o "s" do final das palavras e gerar um novo vetor com as palavras no singular

## Início

O Botão o site é carregado

## Fluxo

1. Criar um vetor global `palavrasPlural = ["Casa", "Livros", "Carros", "Árvores", "Cidades", "Flores"]`;
2. Criar uma função `removerPlural()`:
    2.1 Criar um novo vetor `palavrasSingular`;
    2.2 Guarda, em cada posição de `palavrasSingular`, o conteúdo da mesma posição de `palavrasPlural` substituindo o "s" do final por "";

2.3 Exibe `palavrasPlural`e `removerPlural()`;
