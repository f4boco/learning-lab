# 🤖 Desafio: Ouvidoria de Rastreio

Um sistema logístico recebe um bloco de texto bruto contendo comentários de clientes e códigos de rastreamento misturados (ex: "Olá, meu produto não chegou. Meu código é BR123456789X. O do meu amigo é BR987654321X").
Os códigos de rastreio da empresa sempre seguem o padrão fixo: duas letras maiúsculas, seguidas por 9 números, seguidas pela letra X.

Seu script deve analisar esse texto e extrair todos os códigos de rastreio que encontrarem correspondência com esse padrão exato, exibindo-os em uma lista na tela.
