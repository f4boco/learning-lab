# 🤖 Desafio: Localizador de Emails Corporativos

O programa recebe um bloco de texto contendo vários endereços de e-mail misturados. O script deve extrair apenas os e-mails que pertencem ao domínio corporativo da empresa, ou seja, aqueles que terminam exatamente com @empresa.com.br.

Resultado: Exiba a lista de e-mails corporativos encontrados na tela.
